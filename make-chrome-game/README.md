# :dragon_face: 크롬 공룡게임 만들기

### 📌 프로젝트 소개
Canvas와 requestAnimationFrame()를 이용해 장애물을 넘는 애니메이션을 제작한 프로젝트

<br>

### 📌 기능 구현
- 선인장이 무한으로 왼쪽으로 이동되는 모션
- 스페이스 바를 누르면 공룡이 점프하는 모션
- 공룡과 선인장이 충돌했을 때의 처리
